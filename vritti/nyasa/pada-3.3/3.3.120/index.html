3.3.120 अवे तृ?स्त्रोर्घञ्।

स्तृणातेःइति। स्तृञ् आच्छादने (धा।पा।१४८४) इत्यस्मात्। प्रायानुवृत्तेः इत्यादि। प्रायशब्दस्यानुवृत्तौ सत्यामेव उपपदे तृस्त्रोर्घञ्प्रत्ययो भवति संज्ञायां प्रायेणोपधिभूतायामित्येषोऽर्थो लभ्यते। `न संज्ञायाम्िति। अस्योपाधेः प्रायिकत्वात् क्वचिदसंज्ञायामपि भवति॥
