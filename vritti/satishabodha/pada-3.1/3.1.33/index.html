3.1.33 स्यतासी लृलुटोः

वृत्तिः – धातोः “स्य” “तासि” एतौ प्रत्ययौ स्तो लृलुटोः परतः । The affixes “स्य” and “तासिँ” are prescribed after a धातुः when followed by “लृँ” (लृँट् or लृँङ्) or लुँट् respectively. Note: This rule is a अपवाद: for 3.1.68 etc.

Example continued from 3.3.15

भू + ति

= भू + तासिँ + ति 3.1.33

= भू + तास् + ति 1.3.2, 1.3.9

= भू + इट् तास् + ति 7.2.35, 1.1.46

= भू + इतास् + ति 1.3.3, 1.3.9

= भो + इतास् + ति 7.3.84

= भवितास् + ति 6.1.78
